# SHK Bot (MAX)

Бот для MAX (ex. Telegram) для сканирования и генерации штрих-кодов и QR-кодов.

## Возможности

- **Читать штрих-коды** -- отправь фото, бот распознает ШК/QR
- **Создавать штрих-коды** -- отправь текст, получи Code128
- **OCR fallback** -- если штрих-код не найден, распознаёт текст
- **Защита от спама** -- rate limiting между запросами

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить токен
cp .env.example .env
# Вписать MAX_BOT_TOKEN из business.max.ru

# 3. Запустить
python main.py
```

## Конфигурация

| Переменная | По умолчанию | Описание |
|---|---|---|
| `MAX_BOT_TOKEN` | *обязательно* | Токен бота из MAX |
| `OCR_API_KEY` | `helloworld` | Ключ OCR.space |
| `RATE_LIMIT_SECONDS` | `2` | Интервал между запросами |
| `MAX_FILE_SIZE_MB` | `20` | Макс. размер файла |
